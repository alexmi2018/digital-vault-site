Support

Digital Vault Support

For help, contact support@example.com.

How to report a problem

Send a short description of the issue, what you were trying to do, and your device model and Android version if you know them.

Backup and restore help

If you use Google Drive backup, make sure you are signed in to the same Google account and have allowed the app to access Drive when prompted.

Deleting local data

To remove local data, uninstall the app or clear the app's data in Android settings.

Deleting Google Drive backup files

You can delete backup files directly from your Google Drive if you no longer want them stored there.

Pro features and purchase support

If you need help with Pro features or a purchase, include the email address used for the purchase and a brief description of the issue when you contact support.
